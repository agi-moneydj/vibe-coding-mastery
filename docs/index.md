## 關於本指南

本指南是《**Vibe Coding: Building Production-Grade Software with AI**》（作者：Gene Kim & Steve Yegge，2025）的逐章精通筆記。

**Target Audience**: 有使用 Claude Code、GitHub Copilot 或 Gemini 經驗的 RD

**重點聚焦**:
- 提升開發**速度**、**正確性**、**效率**
- Code Review 策略
- Testing（Unit Testing、E2E Testing、AI 時代的測試新思維）
- 架構圖以 ASCII ART 呈現核心概念

> *「No one should be writing code by hand anymore if they don't have to.」— Gene Kim & Steve Yegge*
